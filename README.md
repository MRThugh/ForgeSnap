# ForgeSnap

![ForgeSnap](https://github.com/MRThugh/ForgeSnap/blob/main/assets/banner.png)

**ForgeSnap** is a beautiful, fast, and fully offline code screenshot tool built with Python and PySide6.  
Inspired by Carbon.sh and Ray.so — but with more control, extra features, and a polished desktop experience.

---

## ✨ Features

- **High-quality rendering** up to 8× (Ultra HD)
- **Multiple language support** (Python, JavaScript, HTML, CSS, JSON, C++, Java, etc.)
- **Popular themes** — Dracula, Monokai, GitHub Dark, Nord
- **Beautiful gradient backgrounds**
- **Preset canvas sizes** — 16:9, 1:1 Square, 9:16 Story, 4:5 Portrait, Auto
- **Line numbers** + Custom watermark
- **Auto-save settings** using QSettings
- **Drag & Drop** code files with automatic language detection
- **Custom neon app icon** generated with Pillow
- **Smart caching** for buttery-smooth live preview
- **Fully offline** — no account or internet required

## 🚀 Quick Start

### Requirements
- Python 3.8+
- PySide6, Pillow, Pygments

### Installation

```bash
git clone https://github.com/MRThugh/ForgeSnap.git
cd ForgeSnap

pip install -r requirements.txt

python main.py
```

### `requirements.txt`
```txt
PySide6>=6.5.0
Pillow>=10.0.0
Pygments>=2.15.0
```

---

## 📖 How to Use

1. Paste your code or **drag & drop** a file into the window.
2. Choose language, theme, canvas size, gradient, etc.
3. See real-time live preview.
4. Click **Copy Image** or **Export PNG**.
